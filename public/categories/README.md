# Category photos

Drop one professional photo per **category type** here and it appears on every
matching car card across the marketplace — no matter how each rental company
spells their category. Until a file is added, that category falls back to the
built-in illustration, so nothing looks broken.

## Filenames the site looks for (lowercase, `.jpg`)

| File | Used for category names like… |
|---|---|
| `economy.jpg` | Economy, Economy plus, Student, Sub-Compact, Mini, Budget, Small |
| `compact.jpg` | Compact, Compact automaat, Compact Car, Compact handgeschakeld |
| `hatchback.jpg` | Hatchback |
| `sedan.jpg` | Sedan, Mid-Size / Midsize, Mid Class Sedan, Intermediate, Standard, Full-size |
| `suv.jpg` | SUV, Mini SUV, Mid-size SUV, Full-size SUV, Jeep, 4x4, Crossover |
| `pickup.jpg` | Pickup, Pickup truck, Truck |
| `van.jpg` | Van, Minivan, MPV, 7/8-seater |
| `luxury.jpg` | Luxury, Premium, Executive |
| `convertible.jpg` | Convertible, Cabrio |

You only need the ones you actually rent. Today the fleets use mostly
**economy, compact, sedan and suv** — start with those four.

## Photo tips

- **Landscape**, around **1200 × 750px** (cards are wide, ~16:10).
- **JPG**, compressed to **under ~250 KB** each (squoosh.app / tinypng.com) so
  cards load fast.
- A clean studio/3-4 view of the car type on a simple background looks most
  professional and consistent across the grid. Keep the framing similar between
  photos so the grid feels uniform.
- No text or watermarks — they clash with the card overlay.

## How to add them

GitHub → this folder → **Add file → Upload files** → drop the images in with the
exact names above → **Commit changes**. They go live on the next deploy.
