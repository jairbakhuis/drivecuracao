# Site images

Drop banner/hero photos in this folder using the **exact filenames** below and
they appear automatically on the matching page (no code changes needed). Until a
file is added, that banner shows a green gradient fallback — nothing breaks.

## Filenames the site looks for

| Filename | Where it shows |
|---|---|
| `hero-home.jpg` | Homepage hero (the big one) |
| `hero-cars.jpg` | "Browse cars" banner |
| `hero-how.jpg` | "How it works" banner |
| `hero-about.jpg` | "About" banner |
| `hero-faq.jpg` | "FAQ" banner |
| `local-panel.jpg` | *(optional)* the "Built in Curaçao" panel on the homepage |

## Tips for good banners

- **Landscape**, at least **2000px wide** (banners are full-width).
- **JPG**, compressed to **under ~500 KB** each (use squoosh.app or tinypng.com)
  so pages load fast — big images hurt SEO and mobile.
- Leave some calm/empty area (sky, water, road) where the headline sits — the
  text goes on the **left**, so an image with open space on the left reads best.
- A dark overlay is applied automatically for text legibility.

## How to add one

GitHub → this folder → **Add file → Upload files** → drop the image in with the
right name → **Commit changes**. That's it.
